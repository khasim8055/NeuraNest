## TARGET CLINIC PROFILE

Best pilot candidates:
- Private GP practice, 1–3 doctors
- Already uses some digital tools (not fully paper-based)
- Discharges 5+ patients per week
- Located in DACH region (Germany, Austria, Switzerland)
- Frustrated with documentation time

Avoid for first pilots:
- Public hospitals (procurement takes 12–18 months)
- Large MVZ (too many stakeholders)
- Fully paper-based practices (too much change management)
